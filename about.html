About the Project

This is web project of L2 Computer Science student - Fahmin Guliyev.In this project, a number of popular languages for web development used such as HTML5 , CSS3 , JavaScript , PHP7 , jpGraph and MySQL phpmyadmin as database.

The website is fully operational and responsive containing a transaction adder page in which you can add a new transaction, see the recently added transactions and even delete an accidentally added transaction from recently added.There is another page where you can see all transactions (database of transactions, search, sort and etc).Other than those, a section called Graphic Analyzer which analyzes the transaction data with the help of php tool jpGraph appears in the navigation of welcome page.

It's very easy and clear to navigate through the pages and use the website.

1.To add some transactions to our database, please go back from this page and navigate to "Add Transaction" section.

2.To see the database and history of transactions, please go back from this page and navigate to "Database" section.

3.To see the graph created by jpGraph using the database, please go back from this page and navigate to "Graph Analyzer" section.
